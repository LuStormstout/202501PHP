文件路径：
「./」 代表当前同级目录
「../」 代表上一级目录
「/」 代表根目录
「~」 代表用户根目录（一般不会用）
「直接写文件名是什么意思」 代表当前同级目录下的文件
绝对路径：从根目录开始写
相对路径：从当前目录开始写
命令行中常用的操作路径的命令：cd ls

This feature is well established and works across many devices and browser versions.
